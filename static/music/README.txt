=====================================
  BRAIN CAPITAL - Music Folder
=====================================

Як додати "Izzamuzzik - Shootout" або іншу музику:

СПОСІБ 1 - Через плеєр на сайті:
--------------------------------
1. Відкрийте сайт
2. Натисніть на плеєр в лівому нижньому куті
3. Натисніть кнопку завантаження (⬆️)
4. Виберіть ваш MP3 файл

СПОСІБ 2 - Додати файл на сервер:
---------------------------------
1. Завантажте трек "Izzamuzzik - Shootout" 
   (можна знайти на YouTube Music, Spotify, etc.)
   
2. Збережіть MP3 файл в цю папку:
   static/music/shootout.mp3

3. Відредагуйте файл templates/base.html
   Знайдіть рядок з <audio id="bgMusic">
   Додайте source:
   
   <source src="/static/music/shootout.mp3" type="audio/mpeg">

4. Перезапустіть сервер

ПРИМІТКА:
---------
Через обмеження браузерів, автоматичне відтворення музики
заблоковане. Користувач повинен натиснути кнопку Play.

Підтримувані формати: MP3, WAV, OGG, M4A
