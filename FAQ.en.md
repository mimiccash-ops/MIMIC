# Frequently Asked Questions

## 1. What is MIMIC?
MIMIC is an automated copy-trading platform. Master trader orders are mirrored to your exchange account.

## 2. How do I start?
1. Register.
2. Add futures API keys (no withdrawals).
3. Wait for approval and start trading.

## 3. Which exchanges are supported?
Binance, Bybit, OKX, Bitget, MEXC and more. The current list is shown during exchange connection.

## 4. What API permissions are required?
- Futures trading: ✅
- Withdrawals: ❌
- IP whitelist: ✅ recommended

## 5. How is risk managed?
You set risk, leverage, TP/SL. Trades are copied using your settings.

## 6. Subscription & payments
Subscription status and options are shown in your dashboard. If subscriptions are disabled by admin, access is free.

## 7. Account recovery
Use the password reset page. Codes are delivered via Email or Telegram (if configured).
