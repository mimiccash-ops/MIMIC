Quick Start

Authenticate by including these headers in your requests: 

# Required Headers
X-API-Key: mk_your_api_key_here
X-Timestamp: 1704067200  # Unix timestamp
X-Signature: hmac_sha256_signature

Example: Submit a trading signal 

POST https://api.mimic.cash/v1/signal

{
  "symbol": "BTCUSDT",
  "action": "open_long",
  "leverage": 10,
  "risk_percent": 2.0,
  "stop_loss": 41000,
  "take_profit": 45000
}
